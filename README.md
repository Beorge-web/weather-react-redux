# Проект: Weather
## Описание

Сайт для просмотра погоды в выбраном городе, данные берутся с 'https://www.weatherapi.com/'

## Технологии

- HTML
- CSS (Flex)
- БЭМ (Nested)
- React JS
- React Redux
- REST api

## Установка и запуск

1. Сколнируйте репозиторий:
`git clone https://github.com/Beorge-web/weather-react-redux.git`
2. Установите NPM пакеты:
`npm install`
3. Запуск проекта:
`npm run start`
